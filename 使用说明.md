# Web 界面使用说明

## 🚨 如果遇到 403 错误或无法访问

### 问题 1：依赖未安装

如果看到 `ModuleNotFoundError`，说明缺少依赖库。

**解决方法：**

1. 先安装 SSL 证书（macOS 常见问题）：
   ```bash
   /Applications/Python\ 3.*/Install\ Certificates.command
   ```
   或者找到你的 Python 安装目录，运行 `Install Certificates.command`

2. 然后安装依赖：
   ```bash
   pip3 install -r requirements.txt
   ```

### 问题 2：端口被占用

如果端口 5000 或 8080 被占用，可以：

1. 修改 `web_app.py` 中的端口号（例如改为 5001）
2. 或者停止占用端口的程序

### 问题 3：macOS 安全限制

如果仍然无法访问，尝试：

1. 使用 `127.0.0.1` 而不是 `localhost`
2. 检查系统防火墙设置
3. 尝试使用不同的端口（如 8000, 5001 等）

## 🚀 启动方式

### 方式一：使用启动脚本（推荐）

```bash
./start_web.sh
```

### 方式二：直接运行

```bash
python3 web_app.py
```

然后在浏览器中打开：`http://127.0.0.1:8080`

## 📝 如果 Web 界面无法使用

你可以直接使用命令行版本：

1. 编辑 `main.py`，填入频道 URL
2. 运行：`python3 main.py`
3. 结果会保存在 `output/` 目录
